Put it Online

After you’ve downloaded the template you’ll probably want to edit the files in order to put you own texts and images into the theme. You will need HTML/CSS skills to do this and a HTML editor like Dreamwaver, Sublime Text, Notepad++, BlueGriffon, KompoZer or Bluefish Editor.

Assuming you already have your domain and hosting :) you will have to use FTP client software like FilleZilla to upload files to your server in public_html directory.

Now you can access your new website using web address in web browser like Internet Explorer, Chrome, Firefox or Safari.

Menu structure and styling

Revelance uses Superfish – jQuery menu plugin for main menu dropdowns. HTML markup is classic unordered list ( with id main_menu ) with nested child lists for dropdowns.
Optional class is current-menu-ancestor on 1st level for highlighting currently open menu item or its ancestor in main level. Also has_children class on any item that has childern items.

Contact Form

Here is explanation of how to make a contact form.

The only things you need to change are in /php/sendmail.php file which handles input and sends mail.
Replace yellow fields with your data and put your password on line 38.

These are default settings for Gmail, if you use the mail services of another provider, maybe you will need to change SMTPSecure ( line 34 ) and Port ( line 36 ). 

                  
                    $message_array = array();
                    if(empty($_POST['name'])) $message_array[] = 'Name';
                    if(empty($_POST['email']) || !filter_var($_POST['email'], FILTER_VALIDATE_EMAIL)) $message_array[] = 'Valid Email';
                    if(empty($_POST['subject'])) $message_array[] = 'Subject';
                    if(empty($_POST['message'])) $message_array[] = 'Message';
                    $body = 'Name: '.$_POST['name'].'';
                    $body .= 'Email: '.$_POST['email'].'';
                    $body .= 'Subject: '.$_POST['subject'].'';
                    $body .= 'Message: '.$_POST['message'].'';
                    if(sizeof($message_array)==1){
                        $message = $message_array[0].' is required field.';
                    }
                    elseif(sizeof($message_array)>1){
                        $message = join(' and ', array_filter(array_merge(array(join(', ', array_slice($message_array, 0, -1))), array_slice($message_array, -1)))). ' are required fields.';
                    }
                    else{
                        $message = '';
                    }


                    if($message=='' && (isset($_POST['action']) && $_POST['action']=='js')){
                        require_once('class.phpmailer.php');
                        include("class.smtp.php");

                        $mail = new PHPMailer();


                        //settings here
                        //it is prepaid for Gmail, just enter your data
                        $mail->IsSMTP();                                    // telling the class to use SMTP
                        $mail->SMTPDebug  = 0;                              // enables SMTP debug information (for testing)
                        $mail->SMTPAuth   = true;                           // enable SMTP authentication
                        $mail->SMTPSecure = "tls";                          // sets the prefix to the server
                        $mail->Host       = "smtp.example.com";             // sets GMAIL as the SMTP server
                        $mail->Port       = 587;                            // set the SMTP port for the GMAIL server
                        $mail->Username   = "user@example.com";             // GMAIL username
                        $mail->Password   = "";                             // GMAIL password
                        $mail->SetFrom("from@example.com", "Mailer");       // set from email address
                        $mail->AddAddress("joe@example.net", "Joe User");   // set destination address
                        $mail->Subject = "Question via Contact Form";       // set subject of email

                        $mail->MsgHTML($body);

                        if(!$mail->Send()) {
                          // echo "Mailer Error: " . $mail->ErrorInfo;
                          echo "Error while sending message...";
                        } else {
                          echo "OK";
                        }
                    }
                    else{
                        echo $message;
                    }

Grid Layout and container

The grid system utilizes 12 columns, making for a 1170px wide container.
For a simple two column layout, create a .row and add the appropriate number of .span* columns. As this is a 12-column grid, each .span* spans a number of those 12 columns, and should always add up to 12 for each row (or the number of columns in the parent). 

<div class="row">
  <div class="span4">...</div>
  <div class="span8">...</div>
</div>

Given this example, we have .span4 and .span8, making for 12 total columns and a complete row.

Use class container to wrap some content with responsive centered container of 1170px width. 

<section id="about_us" class="dm_section_DD">
    <div class="container">
        ... content here ...
    </div>
</section>

SEO Search Engine Optimization

In order to improve your search engine ranking, and thus enable your web site to appear among search results, please don’t forget to change the description and keywords with your own. 

<title>Revelance</title>
<meta name="description" content="Premium Onepage Theme">
<meta name="keywords" content="Some keywords that best describes your site">

As a best practice, the keywords from the meta tag are words that you should use in the texts from the page.
